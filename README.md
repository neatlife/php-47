# php-47
# 传智播客PHP就业班 47期 博客项目

目录结构
-------------------

```
app                       包含MVC和配置文件
  config/                 包含项目配置文件
  model/                  包含前后端模型文件
  view/                   包含前后端视图文件
    frontend/             包含前端视图文件
    backend/              包含后端视图文件
  controller/             包含前后端控制器文件
    frontend/             包含前端控制器文件
    backend/              包含后端控制器文件
core                      包含框架和核心文件
vendor                    包含第三方代码
  smarty/                 包含smarty库核心文件
web                       包含前后端静态资源文件
  backend/                包含后端静态资源文件
  frontend/               包含前端静态资源文件
sql                       包含项目sql文件
```
